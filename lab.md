1. Планування

Мета: створити простий застосунок «Міні-блог», у якому користувач може додавати короткі текстові пости. Кожен пост автоматично отримує дату створення. Публікація назовні не передбачена — записи зберігаються лише локально в межах застосунку.

Функціональність має дозволяти користувачеві швидко створювати записи, переглядати їх у списку та зберігати власну історію думок.

2. Аналіз вимог (User Stories)

Must have:

Як користувач, я хочу додавати короткий пост, щоб зберігати свої думки чи події дня.

Як користувач, я хочу бачити дату створення посту, щоб знати, коли його було додано.

Other:
3. Як користувач, я хочу переглядати всі мої пости у списку, щоб швидко знаходити попередні записи.
4. Як користувач, я хочу мати можливість видаляти пост, щоб прибирати непотрібні записи.
5. Як користувач, я хочу очищати весь міні-блог, щоб почати заново.

3. Дизайн (Прототип)
Головний екран

Назва: “Mini Blog”

Список постів (картки: текст + дата)

Кнопка “+” — додати новий пост

Екран додавання поста

Поле: “Enter your text…”

Кнопка “Save”

Кнопка “Cancel”



4. Реалізація (Псевдокод)

Алгоритм додавання поста

function addPost(text):
    if text is empty:
        return "Error: Post cannot be empty"

    post = {
        "content": text,
        "date": getCurrentDate()
    }

    postsList.add(post)
    return "Post added successfully"

5. Тестування
Тест 1 — Додавання валідного посту

Вхід: "Сьогодні я почав вести міні-блог"
Очікування: Пост з’являється у списку з датою.

Тест 2 — Додавання порожнього посту

Вхід: ""
Очікування: Система повертає помилку "Error: Post cannot be empty".

Тест 3 — Додавання двох постів підряд

Вхід: Пост 1 — “Почав день з кави”, Пост 2 — “Вивчаю SDLC”.
Очікування: У списку два пости, кожен із власною датою та в правильному порядку.

6. Висновки

Для цього проєкту найкраще підходить модель Agile, оскільки застосунок невеликий, простий, легко розширюється та може швидко змінювати функціональність залежно від потреб користувача.
Waterfall був би надто жорстким, а Spiral — надто складним для такого масштабу.
